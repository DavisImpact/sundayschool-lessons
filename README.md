# Sunday School Lessons – Aiea Ward (Facilitator Resources)

This is a Jekyll-powered site containing facilitator notes for the Adult Sunday School class.

## Current Lesson
- [The Battle Is the Lord’s](facilitator/2026-06-21) – June 21, 2026

## Site Structure
- `index.md` – Landing page
- `facilitator/` folder – Contains weekly facilitator notes as Markdown files (e.g. `2026-06-21.md`)
- `facilitator/_template.md` – Template for creating new weekly lessons

## How to Add a New Week
1. Copy `facilitator/_template.md`
2. Rename it to `facilitator/YYYY-MM-DD.md`
3. Update the front matter and content
4. Update `index.md` to link to the new lesson
