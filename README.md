## How to Add a New Week
1. Create a new file in `facilitator/YYYY-MM-DD.md`
2. Use the front matter:
   ```yaml
   ---
   layout: default
   title: "Lesson Title - Facilitator Notes"
   date: YYYY-MM-DD
   ---
   ```
3. Update `index.md` to link to the new lesson.
