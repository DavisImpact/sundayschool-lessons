# Sunday School Lessons – Aiea Ward (Facilitator Resources)

This Jekyll site contains facilitator notes for the Adult Sunday School class in the Aiea Ward.

## Current Lesson
- [The Battle Is the Lord’s](facilitator/2026-06-21) – June 21, 2026

## How to Add a New Week (Recommended)

1. Copy the template file: `facilitator/_template.md`
2. Rename it to `facilitator/YYYY-MM-DD.md` (e.g., `facilitator/2026-06-28.md`)
3. Update the front matter:
   ```yaml
   ---
   layout: default
   title: "[Lesson Title] - Facilitator Notes"
   date: YYYY-MM-DD
   scripture: "[Scripture Reference]"
   ---
   ```
4. Fill in the sections with discussion prompts and application questions.
5. Update `index.md` to link to the new lesson.

## Template
A ready-to-use template is available at `facilitator/_template.md`.

## Philosophy
This site supports a facilitator approach: helping class members share what they learned, felt, and were prompted to change from their personal study.
